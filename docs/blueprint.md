# **App Name**: An Lạc Calendar

## Core Features:

- Dynamic Lunar-Solar Dashboard: A high-fidelity landing page displaying real-time Solar/Lunar dates, Can Chi, and seasonal background animations.
- Hybrid Calendar Views: Seamlessly switch between grid, timeline, and agenda layouts with specific markers for lunar transitions.
- AI Destiny Advisor: An AI tool that reasons through Can Chi, Five Elements, and Tiết Khí to suggest auspicious dates for important events like weddings or openings.
- Comprehensive Cosmic Detail: Deep-dive view into daily astrological markers including Zodiac hours, 28 Mansions (Nhị Thập Bát Tú), and Auspicious/Inauspicious omens.
- Personal Event Management: Secure storage of reminders and events using a PostgreSQL database with cross-platform synchronization.
- Vietnamese Cultural Catalog: Curated repository of traditional festivals, solar terms (24 Tiết Khí), and detailed illustrations for each.
- Native PWA Experience: Mobile-first PWA features including push notifications for auspicious times and offline mode capability.

## Style Guidelines:

- Primary Color: Crimson Red (#E53935), evoking tradition, prosperity, and vital energy found in Vietnamese festivals.
- Background Color: Sophisticated Pearl Gray (#F8F4F4), a desaturated, high-brightness tint of the primary to ensure a clean, iOS-style editorial feel.
- Accent Color: Golden Amber (#FFD54F), providing high contrast for Zodiac hour highlights and primary actions.
- Headline and Numeric font: 'Inter' for precise, modern legibility. Body font: 'Be Vietnam Pro' for a local, high-quality editorial feel. Note: currently only Google Fonts are supported.
- Glassmorphic components with subtle blurs and Dynamic Island-style overlays to mimic an iOS 18 native application experience.
- Smooth, fluid transitions between calendar states using Framer Motion to simulate physics-based native interactions.
- Ultra-modern 3D and SVG illustrations inspired by traditional lacquer art, modernized with soft shadows and neon highlights.
